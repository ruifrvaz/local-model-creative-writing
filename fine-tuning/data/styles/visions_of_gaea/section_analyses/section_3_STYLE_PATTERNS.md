## Quick Reference: Core Patterns

### POV Pattern
```
ALWAYS: Second-person "you" for protagonist
FORMAT: "you [past-tense verb] [object/description]"
EXAMPLE: "you felt your skull being crushed"
EXCEPTION: Epilogue shifts to third-person "she" for transcendence
```

### Internal Monologue Pattern
```
FORMAT: ...italicized text...
PLACEMENT: After significant action or for philosophical commentary
TENSE: Present
EXAMPLE: ...not even after a thousand deaths would I spare you from retribution...
```

### Scene Break Pattern
```
MARKER: ***
USE FOR: POV shifts, time jumps, location changes, consciousness shifts
NEVER FOR: Continuous action, minor time gaps, same-scene pauses
```

### Dialogue Pattern
```
FORMAT: '[Text],' [pronoun] [tag].
QUOTES: Single quotes only
TAGS: said, asked, answered, spoke, conveyed, whispered, shouted
TELEPATHY: '[Thought],' you conveyed the thought to [name].
```

## Sentence Structure Patterns

### Combat Rhythm
```
[Short action]. [Short action]. [Compound result].
Then, [finality].
```
**Example:**
> "He coughed once, his lungs drew one last meager breath, and his heart gave out. Then, there was nothingness."

### Social Rhythm
```
[Medium dialogue]. [Action beat]. [Medium dialogue]. [Internal reaction].
```
**Example:**
> "'We are enjoying the view,' Haji explained, 'It is not every day that we are allowed inside a suite.' 'Well, you are supposed to be choosing your companion,' she said, giving him a sly smile."

### Transcendent Rhythm
```
And [verb], [clause], and [clause], and from [source], [sprouted/emerged] [mystical element].
```
**Example:**
> "And as the cracks grew larger, the pitch-black shards that broke free from the walls began to fall into the infinite void, like the last drops of rain in an everlasting night."

### Tag Selection Rules

- **Important statement:** spoke
- **Mind-link:** conveyed the thought
- **Proximity/secrecy:** whispered
- **Distance/urgency:** shouted

## Character Voice Markers

### Alan (Protagonist)
- Short decisive statements
- Impulsive actions
- Questions authority
- Protective instincts
```
Pattern: '[Short statement],' you [action verb].
Example: "'No,' you said imposingly."
```

### Haji (Best Friend)
- Logical explanations
- Formal vocabulary
- Risk assessment
- Longer sentences
```
Pattern: '[Explanation with technical terms],' Haji [reasoned/explained/warned].
Example: "'Entertainment leads to disorder. And that is directly against our Prime Directive.'"
```

### Wollen (Antagonist)
- Rhetorical questions
- Mocking tone
- Crude humor
- Threatening underlayer
```
Pattern: '[Sarcastic question/statement],' he [mocked/sneered/chuckled].
Example: "'Haven't you had enough?' he asked."
```

### Cathy (Romantic)
- Emotional directness
- Vulnerability
- Confession style
- Shorter nervous sentences
```
Pattern: '[Emotional statement],' she [confessed/said with effort].
Example: "'I've always liked you, Alan,' she confessed with some effort."
```

## Worldbuilding Integration Patterns

### Term Introduction
```
[Term] introduced through action context, not definition
```
**Example:**
> "They were Stalkers, droids designed for the sole purpose of capturing darmians who had strayed too far from Darm's directives, and it was the first time you saw them."

- Core terms repeated (Mother, Unit, sentinels)

## Pacing Formulas

### Combat Pacing
```
Sentence length: 5-15 words
Paragraph length: 2-4 sentences
Ratio: 80% action, 20% reaction
```

### Social Pacing
```
Sentence length: 15-30 words
Paragraph length: 1-3 sentences (dialogue)
Ratio: 60% dialogue, 40% action/thought
```

### Introspective Pacing
```
Sentence length: 20-40+ words
Paragraph length: 3-6 sentences
Ratio: 20% action, 80% internal
```

### Transcendent Pacing
```
Sentence length: 30-80+ words
Paragraph length: Extended single paragraphs
Ratio: 100% narration, 0% dialogue
```

## Scene Structure Templates

### Action Scene
```
1. OPENING: Immediate threat/action
2. ESCALATION: Combat beats (short sentences)
3. CLIMAX: Decisive moment
4. RESOLUTION: Aftermath + transition
5. CLOSE: *** or ellipsis commentary
```

### Social Scene
```
1. OPENING: Setting + character positions
2. EXCHANGE: Dialogue with action beats
3. TENSION: Emotional undercurrent rises
4. REVELATION: Key emotional moment
5. CLOSE: Character departure or transition
```

### Transcendent Scene
```
1. OPENING: Abstract phenomenon
2. AWAKENING: Entity awareness
3. TRANSFORMATION: Light/sound imagery
4. INTEGRATION: Becoming whole
5. CLOSE: New state of being
```
